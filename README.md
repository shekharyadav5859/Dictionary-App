 Dictionary App

A simple and responsive **Dictionary Web Application** built using **HTML, CSS, and JavaScript**. Users can search for any English word and instantly get its meaning, part of speech, example sentence, and synonyms using a free dictionary API.

##  Features

*  Search any English word
* Display word meaning
*  Show part of speech
*  Display example sentence (if available)
*  Show synonyms (if available)
*  Responsive design for Desktop, Tablet, and Mobile
*  Modern and clean UI

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Dictionary API

##  API Used

**Free Dictionary API**

https://api.dictionaryapi.dev/api/v2/entries/en/
